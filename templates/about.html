About This Project

Project Overview

The Graphura Intern Portfolio Readiness Scorer is an AI-powered platform designed to help Graphura evaluate how job-ready interns are by analyzing their portfolios, projects, skills, tools, and presentation.

Business Objective

To create a data-driven system that objectively evaluates intern portfolios, identifies top-performing interns, improves training outcomes, and supports hiring, placement, and employer branding initiatives.

Key Features
  • Resume Evaluation: Upload and get instant AI-powered analysis of job readiness
  • Comparison Panel: Compare up to 6 resumes side-by-side
  • Analytics Dashboard: View key metrics and department-wise performance
  • Leaderboard: See top-performing interns
  • Personalized Recommendations: Get actionable feedback for improvement
How It Works
  1. Resume text is extracted using PDF parsing libraries
  2. Features are extracted (projects, skills, experience, certifications, etc.)
  3. ML model predicts readiness score and classification
  4. Personalized recommendations are generated
Technology Stack
  • Backend: Flask (Python)
  • ML Framework: Scikit-learn, Random Forest
  • PDF Processing: pdfplumber, PyPDF2
  • Data Processing: Pandas, NumPy
  • Frontend: Bootstrap 5, Chart.js
  • Visualization: Chart.js for interactive charts
  • Model Serialization: Joblib
